## بپرس چیه؟

**بپرس** یه ربات تلگرامی هوشمنده که پشتش از مدل **Gemini 2.5 Flash** گوگل استفاده می‌کنه و می‌تونه نتایج جستجوی **زنده‌ی گوگل** رو هم به جوابش اضافه کنه. یعنی وقتی می‌پرسی «قیمت دلار امروز چنده؟»، بپرس می‌ره گوگل سرچ می‌کنه، چند تا منبع معتبر رو می‌خونه، و یه جواب جمع‌بندی شده با لینک منابع برات می‌فرسته.

ربات به‌طور پیش‌فرض **به فارسی جواب می‌ده** ولی هر زبانی بهش بنویسی، به همون زبان جواب می‌گیری.

سه ویژگی اصلی که بپرس رو از بقیه‌ی ربات‌های مشابه جدا می‌کنه:

1. 🔓 **متن‌باز و قابل میزبانی روی سرور خودت** — با یه دستور می‌تونی روی سرور شخصی‌ت اجراش کنی
2. 🔒 **هیچ سوالی ذخیره نمی‌شه** — نه متن سوالت، نه جوابش، حتی توی لاگ‌های سرور هم نمی‌ره
3. 🌐 **بدون فیلترشکن کار می‌کنه** — چون از داخل تلگرام استفاده می‌کنی، نیازی به VPN نداری

## چرا بپرس؟

اگه توی ایران زندگی می‌کنی، احتمالاً می‌دونی که دسترسی به ابزارهای پایه‌ی اینترنتی چقدر سخت شده:

- **گوگل سرچ** فیلتر شده. باید VPN روشن کنی فقط برای یه سوال ساده
- **چت‌جی‌پی‌تی، کلاد، پرپلکسیتی** هم بدون VPN در دسترس نیستن
- **VPN** خودش کند، گرون و ناپایداره

ولی **تلگرام** هنوز از طریق پروکسی‌های MTProto راحت کار می‌کنه و توی ایران به‌طور گسترده استفاده می‌شه. بپرس از همین استفاده می‌کنه: تو از تلگرام می‌پرسی، سرور بپرس (که تو ایران نیست) با گوگل حرف می‌زنه، جواب رو از تلگرام می‌گیری.

نتیجه: **بدون فیلترشکن، بدون پرداخت اشتراک، با حریم خصوصی کامل، به یه ابزار سرچ هوشمند دسترسی داری.**

## شروع سریع — استفاده از ربات

اگه فقط می‌خوای از یه نسخه‌ی موجود استفاده کنی (نه اینکه خودت میزبانی کنی):

1. تلگرام رو باز کن
2. اسم ربات رو که از سازنده‌ش گرفتی توی جستجو بزن (مثلاً `@Bepors_iraneman_bot`)
3. روی **Start** بزن یا `/start` بفرست
4. سوالت رو تایپ کن و بفرست — همین.

پیشنهاد می‌کنم اول از این مثال‌ها استفاده کنی تا ببینی چطور کار می‌کنه:

| نوع سوال | مثال |
|---|---|
| 💵 قیمت ارز | «قیمت دلار امروز چقدره؟» |
| 🌤 آب و هوا | «هوای تهران فردا چطوره؟» |
| 📰 خبر | «آخرین خبر مهم ایران چیه؟» |
| 🏥 سلامت | «بهترین درمان سرماخوردگی چیه؟» |
| 💻 برنامه‌نویسی | «چطوری توی پایتون یه فایل JSON بخونم؟» |
| 📚 دانش عمومی | «پایتخت استرالیا کجاست؟» |
| 🧠 توضیح مفاهیم | «هوش مصنوعی چطور کار می‌کنه؟» |

## دستورات کامل

این‌ها همه‌ی دستورهای ربات هستن. می‌تونی روی دکمه‌ی `/` (منو) توی تلگرام تپ کنی تا لیست‌شون رو ببینی.

### دستورات عمومی (همه می‌بینن)

| دستور | کاری که می‌کنه |
|---|---|
| `/start` | شروع گفت‌وگو + نمایش مثال‌های پیشنهادی |
| `/help` | راهنمای داخل ربات (نسخه‌ی خلاصه‌تر این فایل) |
| `/usage` | نشون می‌ده امروز چندتا سوال پرسیدی و چندتا برات مونده |
| `/lang` | تغییر زبان ربات (فارسی / انگلیسی) |
| `/search on\|off` | جستجوی زنده‌ی گوگل رو روشن/خاموش می‌کنه |
| `/filters` | فیلترهای پیش‌فرض جستجو رو با دکمه تنظیم می‌کنی |
| `/format` | حالت نمایش پاسخ رو عوض می‌کنه (فشرده / تفصیلی) |
| `/privacy` | سیاست حریم خصوصی |
| `/forget` | همه‌ی اطلاعات تو رو از دیتابیس پاک می‌کنه (با تأیید) |

### محدودیت‌ها

- **روزانه ۲۰ سوال رایگان** برای هر کاربر (قابل تغییر توسط ادمین)
- **دقیقه‌ای حداکثر ۵ پیام** (محدودیت سرعت)
- اگه خیلی سریع پشت‌هم پیام بفرستی، یه ساعت صبر می‌کنه

## میانبرها و فلگ‌های جستجو

### میانبرهای ابتدای پیام

- **پیام رو با `?` شروع کن** → جستجوی زنده فقط برای همین یک پیام **روشن** می‌شه
  - مثال: `? آخرین قیمت بیتکوین`

- **پیام رو با `.` شروع کن** → جستجوی زنده فقط برای همین یک پیام **خاموش** می‌شه
  - مثال: `. یه شعر کوتاه از حافظ بنویس` (نیازی به سرچ نیست، مستقیم از مدل می‌گیریم)

### فلگ‌های داخل پیام (وقتی جستجو روشنه)

می‌تونی توی هر پیام از این فلگ‌ها استفاده کنی تا جستجو رو محدود کنی:

| فلگ | کاربرد | مثال |
|---|---|---|
| `--site:domain.com` | جستجو فقط توی یه سایت خاص | `قیمت دلار --site:tgju.org` |
| `--time:day` | فقط نتایج ۲۴ ساعت اخیر | `اخبار ایران --time:day` |
| `--time:week` | فقط نتایج هفته اخیر | `--time:week` |
| `--time:month` | فقط نتایج ماه اخیر | `--time:month` |
| `--time:year` | فقط نتایج سال اخیر | `--time:year` |
| `--lang:fa` | منابع فارسی | `--lang:fa` |
| `--lang:en` | منابع انگلیسی | `--lang:en` |
| `--region:ir` | منابع از یه منطقه‌ی خاص | `--region:ir` |
| `--news` | میانبر برای خبر روز (معادل `--time:day` با تمرکز روی منابع خبری) | `جنگ غزه --news` |
| `--academic` | منابع علمی و دانشگاهی | `یادگیری عمیق --academic` |

می‌تونی چند تا فلگ رو با هم ترکیب کنی:

```
آخرین خبر اقتصاد ایران --site:tasnimnews.com --time:day
```

### فیلترهای پیش‌فرض با `/filters`

اگه نمی‌خوای هر بار فلگ تایپ کنی، می‌تونی فیلترهای پیش‌فرض تنظیم کنی:

1. `/filters` رو بفرست
2. روی دکمه‌ها تپ کن تا فیلترها رو روشن/خاموش کنی
3. تنظیمات ذخیره می‌شه و روی همه‌ی سوال‌های بعدی‌ت اعمال می‌شه

برای پاک کردن همه‌ی فیلترها روی دکمه‌ی **❌ پاک‌کردن همه** بزن.

## حالت‌های نمایش پاسخ

با `/format` می‌تونی بین این دو حالت سوییچ کنی:

### فشرده (پیش‌فرض)
خلاصه‌ی جواب در یه پاراگراف. بهترین حالت برای موبایل.

### تفصیلی
جواب رو به سه بخش تقسیم می‌کنه:
- 📌 **خلاصه** — یه پاراگراف اول
- 📖 **جزئیات** — توضیح کامل‌تر
- 🔗 **منابع** — لینک‌های منبع

اگه جواب شامل ارقام (مثل قیمت، آمار، دما) باشه، ربات به‌طور خودکار با ایموجی‌های `💰` `📈` `🌤` خوانایی رو بهتر می‌کنه و اعداد رو هم به فارسی تبدیل می‌کنه (`۸۲٬۳۰۰` به‌جای `82,300`).

## حریم خصوصی

این بخش مهم‌ترین قسمت پروژه‌ست. اگه نمی‌خوای کسی بدونه چی می‌پرسی، بپرس برای همین ساخته شده.

### چیزی که ذخیره می‌کنیم

| داده | چرا |
|---|---|
| شناسه‌ی عددی تلگرامت (مثلاً `99903888`) | برای شمارش سهمیه‌ی روزانه و جلوگیری از سوءاستفاده |
| زبان انتخابی، حالت نمایش، فیلترهای پیش‌فرض | برای اینکه دفعه‌ی بعد همون تنظیمات رو داشته باشی |
| تعداد سوال‌های امروزت (فقط شمارنده) | برای محاسبه‌ی سهمیه |
| نوع رویداد (پرسش / جواب / خطا) + طول سوال + زمان پاسخ | برای پایش سلامت سیستم — **نه متن سوال، نه متن جواب** |

### چیزی که هرگز ذخیره نمی‌کنیم

- ❌ **متن سوال‌هات**
- ❌ **متن جواب‌هات**
- ❌ **نام، شماره تلفن، ایمیل، یا هر اطلاعات شخصی دیگه‌ای**
- ❌ **تاریخچه‌ی گفتگو**

شناسه‌های کاربری توی لاگ‌ها هم به‌صورت **هش‌شده با SHA-256** و یه نمک تصادفی منحصربه‌فرد ذخیره می‌شن، نه به صورت خام.

### پاک کردن کامل اطلاعاتت

هر وقت خواستی، فقط `/forget` رو بفرست. یه دکمه‌ی تأیید نشونت می‌ده، روی **✅ آره، پاک کن** بزن، و همه‌ی اطلاعاتت توی همون لحظه از دیتابیس حذف می‌شه. برگشت‌پذیر نیست.

### چرا باید بهمون اعتماد کنی؟

دو دلیل:

1. **کد متن‌باز** — کل کد سرور توی [گیت‌هاب](https://github.com/MohammadShamchi/bepors-bot) موجوده. خودت یا یه دوست برنامه‌نویس می‌تونین چک کنین که هیچ‌جا متن سوال ذخیره نمی‌شه.
2. **می‌تونی نسخه‌ی خودت رو روی سرور خودت اجرا کنی** — اگه به نسخه‌ی عمومی اعتماد نداری، با یه دستور می‌تونی نسخه‌ی شخصی‌ت رو راه بندازی. اون‌وقت تنها کسی که داده‌ها رو می‌بینه خودتی.

## راه‌اندازی نسخه‌ی خودت روی سرور

این کار **دو دقیقه** طول می‌کشه. به اینا نیاز داری:

1. **یه سرور لینوکس کوچیک** — هتزنر CX22 ($۴ در ماه)، DigitalOcean ($۵)، Vultr، یا هر چیز مشابه. اوبونتو ۲۲.۰۴ یا ۲۴.۰۴ توصیه می‌شه. سرور باید **خارج از ایران** باشه.
2. **یه توکن ربات تلگرام** — رایگان، از [@BotFather](https://t.me/BotFather) می‌گیری
3. **یه کلید Gemini API** — رایگان، از [Google AI Studio](https://aistudio.google.com/api-keys) می‌گیری (نیاز به VPN داری برای ساخت کلید، فقط یه بار)

### مرحله ۱: SSH به سرور

```bash
ssh root@<آی‌پی-سرور-تو>
```

### مرحله ۲: اجرای دستور نصب

این دستور رو کپی کن و توی ترمینال SSH بچسبون:

```bash
curl -fsSL https://raw.githubusercontent.com/MohammadShamchi/bepors-bot/main/install.sh | bash
```

اسکریپت نصب به ترتیب این کارها رو می‌کنه:

1. ابزارهای لازم رو نصب می‌کنه (`git`, `curl`, `whiptail`)
2. کد رو از گیت‌هاب می‌گیره و توی `/opt/bepors-bot` می‌ذاره
3. یه کاربر مخصوص (`bepors`) با دسترسی محدود می‌سازه
4. محیط مجازی پایتون و وابستگی‌ها رو نصب می‌کنه
5. سرویس systemd رو راه‌اندازی می‌کنه (با تنظیمات امنیتی سخت‌گیرانه)
6. UFW رو فعال می‌کنه (فقط پورت ۲۲ باز)
7. کرون پشتیبان‌گیری روزانه رو نصب می‌کنه (با نگه‌داری ۱۴ روز)
8. آپدیت خودکار امنیتی اوبونتو رو فعال می‌کنه

### مرحله ۳: پنجره‌های تنظیم اعتبارها

بعد از مرحله‌ی ۲، یه سری پنجره‌ی آبی (whiptail) بهت نشون داده می‌شه:

1. **توکن ربات تلگرام** — توکنی که از @BotFather گرفتی رو بچسبون. ربات با Telegram API چک می‌کنه که توکن معتبر باشه. اگه اشتباه باشه، دوباره ازت می‌پرسه.
2. **آی‌دی ادمین (اختیاری)** — اگه می‌خوای دستورات ادمین (`/stats`, `/users`, `/broadcast`, ...) برات فعال بشه، آی‌دی عددی تلگرامت رو وارد کن. (آی‌دی‌ت رو می‌تونی از [@userinfobot](https://t.me/userinfobot) بگیری.) می‌تونی خالی بذاری و بعداً اضافه کنی.
3. **کلید Gemini API** — کلیدی که از Google AI Studio گرفتی رو بچسبون.

تمام. وقتی پنجره‌ی **✓ Installation Complete!** رو دیدی، ربات‌ت آماده‌ست. برو توی تلگرام، اسم ربات رو که موقع ساختش به BotFather دادی پیدا کن، و `/start` بفرست.

### اگه دوباره اجراش کنی

اگه می‌خوای ربات رو به آخرین نسخه ارتقا بدی یا کلیدها رو عوض کنی، **همون دستور رو دوباره اجرا کن**. اسکریپت نصب:

- آخرین نسخه از `main` رو می‌گیره
- اطلاعات موجود (`data/bepors.db`, `.env`) رو دست‌نخورده نگه می‌داره
- پنجره‌ی کلیدها رو با مقادیر فعلی **پر شده** نشون می‌ده، فقط کافیه Enter بزنی یا اگه می‌خوای عوض کنی

یعنی همین یه دستور هم نصب می‌کنه و هم برای ارتقا/تنظیم مجدد استفاده می‌شه.

### بررسی وضعیت ربات

```bash
ssh root@<آی‌پی-سرور-تو>
systemctl status bepors-bot       # وضعیت سرویس
journalctl -u bepors-bot -f       # لاگ‌های زنده (JSON)
curl 127.0.0.1:8088/health        # health endpoint
```

## دستورات ادمین

اگه آی‌دی‌ت رو موقع نصب توی `ADMIN_IDS` گذاشته باشی، توی منوی `/` تلگرام **علاوه بر ۹ دستور عادی، ۷ دستور مدیریتی** هم می‌بینی. کاربرهای دیگه این دستورات رو نمی‌بینن (با `BotCommandScopeChat` به چت تو محدود شده).

| دستور | کاری که می‌کنه |
|---|---|
| `/stats` | آمار کلی: تعداد کاربران امروز، تعداد سوال‌ها، خطاها، استفاده از ظرفیت سراسری |
| `/users [N]` | فهرست N کاربر برتر بر اساس کل سوال‌های پرسیده‌شده (پیش‌فرض ۲۰) |
| `/user <id>` | جزئیات کامل یه کاربر: زبان، حالت، فیلترها، تعداد سوال‌ها، آخرین فعالیت، وضعیت |
| `/broadcast <پیام>` | یه پیام رو به همه‌ی کاربرها می‌فرسته (با محدودیت ۲۵ پیام در ثانیه برای جلوگیری از بلاک شدن) |
| `/block <id>` | یه کاربر رو مسدود می‌کنه |
| `/unblock <id>` | یه کاربر رو از حالت مسدود درمی‌آره |
| `/setlimit <عدد>` | سقف روزانه‌ی پیام برای هر کاربر رو در زمان اجرا عوض می‌کنه |

## سوال‌های پرتکرار (FAQ)

### آیا واقعاً رایگانه؟
بله. هم استفاده از یه نسخه‌ی موجود رایگانه (تا سقف روزانه)، هم میزبانی نسخه‌ی خودت تقریباً رایگانه. هزینه‌ها:
- **سرور**: ۴-۵ دلار در ماه (هتزنر / Vultr)
- **Gemini API**: لایه رایگان گوگل سخاوتمندانه‌ست — اگه کمتر از ~۱۰۰۰ سوال در روز داری، احتمالاً صفر دلار

### آیا می‌تونم از این ربات توی گروه استفاده کنم؟
نسخه‌ی فعلی فقط چت خصوصی (DM) رو پشتیبانی می‌کنه. این یه تصمیم آگاهانه‌ست برای جلوگیری از اسپم گروهی. می‌تونی این رفتار رو با تغییر یه خط توی `bot.py` عوض کنی (دنبال `ChatType.PRIVATE` بگرد).

### آیا بپرس می‌تونه عکس یا ویس بفرسته/بگیره؟
خیر، نسخه‌ی فعلی فقط متنه. ممکنه توی نسخه‌های بعدی اضافه بشه.

### چرا بپرس به جواب من از یه سایت خاص استفاده نکرد؟
گوگل سرچ گراندینگ یه سرویس مستقل از ربات ماست. ما به مدل می‌گیم «از این سایت استفاده کن» ولی مدل ممکنه گاهی نتونه. اگه می‌خوای حتماً از یه سایت خاص بیاد، فلگ `--site:` رو امتحان کن و اگه نتیجه نداد، همون سایت رو مستقیم توی گوگل سرچ کن.

### پاسخ‌های بپرس قابل اعتمادن؟
بپرس از Gemini 2.5 Flash گوگل استفاده می‌کنه که یه مدل قدرتمنده، ولی **هیچ مدل زبانی ۱۰۰٪ دقیق نیست**. همیشه برای موضوع‌های مهم (مثل پزشکی، حقوقی، مالی) به منابعی که زیر جواب لیست می‌شه نگاه کن و خودت دوبار چک کن.

### اگه ربات جواب نداد یا کار نکرد چی؟
اول `/help` رو بفرست تا مطمئن بشی ربات هنوز جواب می‌ده. اگه اون هم جواب نداد، به ادمین ربات (اگه نسخه‌ی عمومی استفاده می‌کنی) یا به خودت (اگه نسخه‌ی شخصی داری) خبر بده.

برای میزبان‌ها: لاگ‌ها رو با `journalctl -u bepors-bot -n 50` چک کن. معمولاً مشکل از کلید API منقضی شده یا سهمیه‌ی Gemini تموم شده‌ست.

### آیا من به‌عنوان ادمین می‌تونم متن سوال‌های کاربرها رو ببینم؟
**نه.** این یه ویژگی به‌عمد طراحی‌شده‌ست. حتی به‌عنوان ادمین، توی `/stats`, `/users`, `/user <id>` فقط آمار می‌بینی (تعداد، زبان، فیلترها) — متن سوال یا جواب هیچ‌جا ذخیره نمی‌شه پس قابل مشاهده هم نیست. این چیزی نیست که بتونه با تغییر تنظیمات روشن بشه؛ کد به‌گونه‌ای نوشته شده که اصلاً متن لاگ نمی‌گیره.

### چقدر امنیتی هست؟
برای جزئیات کامل [بخش امنیت توی README انگلیسی](../README.md#-security-checklist-before-going-live) رو ببین. خلاصه:

- سرویس روی سرور با کاربر **غیرروت** (`bepors`) اجرا می‌شه
- یونیت systemd با `ProtectSystem=strict`, `NoNewPrivileges`, `MemoryDenyWriteExecute` و کلی محدودیت دیگه سخت‌سازی شده
- فقط پورت ۲۲ (SSH) از بیرون باز هست (UFW)
- به‌روزرسانی‌های امنیتی اوبونتو خودکار نصب می‌شن
- فایل `.env` با حالت `600` و مالکیت `bepors:bepors` نگه‌داری می‌شه

## مشارکت در پروژه

دوست داری به بهتر شدن بپرس کمک کنی؟ خیلی استقبال می‌کنیم. چند راه:

### ۱. گزارش باگ یا پیشنهاد ویژگی
یه [issue](https://github.com/MohammadShamchi/bepors-bot/issues) باز کن. هر پیشنهادی برام مهمه.

### ۲. ترجمه به زبان‌های دیگه
ربات الان فارسی + انگلیسی داره. اگه می‌تونی به ترکی، عربی، کردی، یا هر زبان دیگه‌ای ترجمه کنی، فقط فایل‌های `locales/fa.json` و `locales/en.json` رو ببین — همه‌ی متن‌ها اون‌جان. PR بفرست.

### ۳. کد
قبل از فرستادن PR:
```bash
./venv/bin/python -m pytest -q     # باید ۸۱ تست پاس بشه
./venv/bin/python -m py_compile *.py
```

قانون‌های مهم:
- **هرگز** متن سوال یا جواب رو توی لاگ یا دیتابیس ننویس. این اصل اول پروژه‌ست.
- متن‌های جدید رو از `i18n.t(key, lang, **kwargs)` رد کن، نه به‌صورت hardcoded
- پاریتی فایل‌های `locales/fa.json` و `locales/en.json` رو حفظ کن (همه‌ی کلیدها توی هر دو زبان)

### ۴. حمایت از پروژه
اگه از ربات استفاده می‌کنی و مفید بود، یه ⭐ توی [گیت‌هاب](https://github.com/MohammadShamchi/bepors-bot) بهم بده. این کمک می‌کنه پروژه دیده بشه.
